How to Use

Create a workspace with a sender email, create a campaign, add cadence steps, upload contacts, generate drafts, approve drafts, then send or dry run by day.

For best deliverability, each workspace sender should use an SES-authenticated address such as first.last@mail.evolutioncrm.us. Replies will route to the matching @evolutioncrm.us inbox if the app reply-to logic is enabled.
