How to Use

Recommended pilot workflow:

    {% if is_admin %}
  1. Create a workspace for the pilot/client.
  2. Add the sender email for that workspace.
  3. Create a pilot user and assign them to that workspace.
  4. Create a campaign under the correct workspace.
    5. {% else %}
  5. Create a campaign in your assigned workspace.
    6. {% endif %}
  6. Open the campaign workspace.
  7. Add at least one email step.
  8. Upload a CSV of contacts.
  9. Generate drafts.
  10. Review, edit, and approve drafts.
  11. Preview or send when ready.

CSV headers should include:

first_name,last_name,email,company,industry,role,website

{% if is_admin %}

Sender emails are set at the workspace level. Each workspace can send from a different AWS SES verified sender.
