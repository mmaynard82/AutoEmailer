# AI Emailer MVP

A demo AI-assisted outbound email campaign builder for CRM-focused outreach.

## Features

- Campaign creation
- CSV contact upload
- HubSpot contact import/export
- AI-personalized email draft generation
- Template-based emails
- Cadence steps
- Draft review and approval
- Dry-run mode
- Demo mode safety lock
- Unsubscribe/suppression handling
- Basic campaign analytics

## Safety

This app should be used only with contacts you are authorized to contact. Demo mode blocks real email sending.
